## 说明内容

焦点感知域机制用于在复杂故障场景中聚焦关键诊断区域。通过分析当前诊断状态和历史轨迹，动态计算各服务的重要性权重，将诊断资源集中在最可能包含根因的区域内，提高诊断效率。

## 主要内容

- **诊断上下文**: 当前诊断状态信息
- **重要性计算**: 基于多种特征计算服务重要性
- **焦点区域**: 确定的优先诊断区域
- **权重分配**: 为不同服务分配诊断优先级
